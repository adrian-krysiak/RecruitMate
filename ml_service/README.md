## Data Sources & Licensing

This project utilizes the **ESCO (European Skills, Competences, Qualifications and Occupations)** dataset to power its Named Entity Recognition (NER) engine.

This project requires the ESCO dataset. Download skills_en.csv from [ESCO website] and place it in data/raw/. Attribution: This service uses the ESCO classification of the European Commission.
